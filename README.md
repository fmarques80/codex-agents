# Codex Governance

Sistema modular de governança para agentes locais do Codex.

## Estrutura

- AGENTS.md → governança principal
- agents/ → especialistas
- rules/ → regras modulares
- playbooks/ → fluxos operacionais

## Objetivo

Padronizar:
- arquitetura
- segurança
- qualidade
- documentação
- orquestração
- onboarding
- revisão
- troubleshooting

## Instalação

```bash
./install.sh
```
